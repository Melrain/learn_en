# Learn EN 产品设计初稿 v1

本文档记录 Learn EN 游戏化闯关式英文口语练习 APP 的产品设计思路，供后续迭代参考。

---

## 一、设计理念

### 核心定位

- **主题冒险世界**：将每个主题（如奥特曼、恐龙、小猪佩奇）视为独立的冒险世界，孩子在地图上逐步闯关。
- **闯关式学习 + AI 辅助**：关卡驱动口语练习，AI 负责鼓励、检查、并给出针对性发音建议。

### 四大设计支柱

| 维度       | 设计                             |
|------------|----------------------------------|
| 世界观     | 主题世界有专属地图、角色、故事线 |
| 进度感     | 关卡地图可视化，走过的路亮起     |
| 成就感     | 星级评价 + 主题卡片收集          |
| 陪伴感     | AI 伙伴全程鼓励与引导            |

## 二、产品架构

### 整体结构

```
首页（选择世界）
  └── 世界选择（12 个主题世界卡片）
        └── 世界地图（关卡节点，类似超级马里奥世界地图）
              └── 关卡（5–8 个口语挑战）
                    └── 单个挑战（录音 → 评测 → 反馈 → 奖励）
```

## 三、游戏化机制

### 关卡系统

每个世界包含 4 个阶段，对应阿里云评测能力：

| 阶段   | 评测类型       | 描述         |
|--------|----------------|--------------|
| 阶段 1 | en.word.score  | 单词关       |
| 阶段 2 | en.word.score  | 短语关       |
| 阶段 3 | en.sent.score  | 句子关       |
| 阶段 4 | en.pred.score  | Boss 关（段落）|

每个阶段 3–5 关，每关 5–8 个挑战。阶段末尾有 Boss 关，需把前面内容串成一小段话。

### 星级与解锁

| 评测分数 | 星级 | 文案     |
|----------|------|----------|
| ≥ 90     | ⭐⭐⭐ | 完美！   |
| ≥ 70     | ⭐⭐  | 不错！   |
| ≥ 50     | ⭐   | 通过！   |
| < 50     | 未通过 | AI 鼓励重试 |

- 1 星即可解锁下一关，降低挫败感
- 支持重复挑战刷星
- 累计星数可解锁隐藏奖励

### 收集系统

- 每个世界 20–30 张主题卡片/贴纸
- 通关后随机掉落 1–2 张
- 三星通关有概率掉落稀有卡片
- 卡片由 AI 按主题生成，使用现有 DashScope + S3 能力

稀有度示例：

- 普通：常规主题插图
- 稀有：带特殊装饰（如金边）
- 传说：独特特效（如 holographic），连续打卡等成就解锁

### 每日任务与连续打卡

| 机制       | 说明                               |
|------------|------------------------------------|
| 每日 3 关  | AI 从未完成/低分关卡中推荐 3 关     |
| 连续打卡   | 7 天 → 稀有卡片；30 天 → 传说卡片   |
| 每周挑战   | 限时关卡，如「用学过的词造故事」   |

## 四、AI 融合方案

### AI 伙伴角色

设计固定 AI 伙伴（如小鹦鹉 Polly），贯穿全流程：

| 场景       | AI 行为                                   |
|------------|--------------------------------------------|
| 进入关卡   | 主题相关引导语 + TTS 朗读 + 播放主题短视频营造氛围 |
| 示范发音   | 播放标准发音，鼓励跟读                     |
| 评测高分   | 根据分数 + 单词生成个性化鼓励 + 播放庆祝动画视频 |
| 评测低分   | 分析评测详情，给出针对性发音建议           |
| 卡关       | 检测连续失败，播放鼓励短视频，建议休息或明天再试 |
| 通关       | 触发奖励动画视频，如「解锁新卡片」         |

技术：DeepSeek 生成文案 + 现有 TTS + 阿里云评测详情 + 图生视频。

### 动态内容编排

- **传统**：预设固定题库，所有人相同
- **本产品**：AI 根据主题、难度、孩子弱点实时生成题目

流程示例：

1. 孩子选择恐龙世界第 2 关
2. LangGraph Agent 调用：`get_practice_history`、`get_question_stats`
3. 分析薄弱音素（如 th、r）
4. 生成 5 道「恐龙主题 + 句子难度 + 强化 th/r」的题目
5. 为每道题生成恐龙主题配图（DashScope）
6. 将配图异步送入图生视频模型，生成 2–4 秒主题短动画，缓存至 S3

### Boss 故事模式

Boss 关为互动口语小故事，由 AI 实时编排：

- AI 按步骤推进剧情，每步需孩子口语输出
- 评测通过则进入下一步
- 低分时给予「提示」而非直接失败
- 关键场景可配 AI 生成插图，并转化为短动画视频推进剧情
- AI 根据剧情节奏智能控制视频播放时机（如转场、高潮、结局）

技术：LangGraph Agent 维护故事状态 + 阿里云评测 + DashScope 配图 + 图生视频。

### AI 视频生成与播放

将现有 AI 生图能力延伸为「图 → 视频」管线，由 AI 控制播放时机，打造沉浸式动画体验。

**生成管线**

```
AI 出题 → DashScope 生成主题配图 → 图生视频模型生成 2–4 秒短动画 → 缓存至 S3 → AI 控制播放
```

**AI 控制播放时机**

AI 伙伴根据学习流程状态决定何时播放视频：

| 触发条件         | 视频类型       | 示例                             |
|------------------|----------------|----------------------------------|
| 进入新世界/关卡  | 主题氛围视频   | 恐龙世界：火山喷发 + 恐龙奔跑   |
| 通关成功         | 庆祝动画视频   | 角色跳舞 + 星星飞出             |
| 三星通关         | 稀有奖励动画   | 卡片翻转 + 闪光特效             |
| 连续失败后重试   | 鼓励短视频     | AI 伙伴加油打气的动画           |
| Boss 剧情推进    | 剧情过场视频   | 故事关键转折的动画演绎           |

**技术选型**

| 维度     | 方案                                             |
|----------|--------------------------------------------------|
| 图生视频 | 阿里通义万相（与现有阿里云生态一致）或 Runway/Pika |
| 视频规格 | 2–4 秒、512×512 或 720p、无声（配合 TTS 播放）   |
| 生成策略 | 异步预生成：进入关卡前触发，练习时已就绪         |
| 缓存     | 生成后存储至 S3，相同题目复用，避免重复生成       |
| 降级     | 视频未就绪时回退为静态配图，不阻塞学习流程       |

### 新增数据模型

**UserProgress**（用户进度）

```
- userId
- avatar
- totalStars
- streak（连续打卡天数）
- lastPracticeDate
- worldProgress: { themeId: { unlocked, levels: [{ levelId, stars, completed, bestScore }] } }
```

**CollectionCard**（收藏卡片）

```
- userId
- themeId
- imageUrl
- rarity: common | rare | legendary
- earnedAt
- earnedFromLevel
```

**Level**（关卡定义）

```
- themeId
- stage: 1 | 2 | 3 | 4
- order
- title
- description
- challenges（Question ID 数组）
- bossMode?: boolean
```

## 六、设计原则（面向儿童）

1. **永远不说「你失败了」**：低分时说「再试一次，你可以的」，并给出具体改进方向
2. **视觉优先，文字最少**：关键操作用图标 + 语音引导
3. **按钮要大，操作要少**：每个页面核心操作只有一个大按钮
4. **奖励要即时、频繁**：每次录音后立刻有动画反馈（星星、角色动效）
5. **允许自由探索**：除关卡顺序外，可自由选择喜欢的世界和关卡
6. **家长可控**：设置每日练习上限、查看进步报告

## 七、差异化总结

| 维度   | 市面产品（如多邻国）   | Learn EN                     |
|--------|------------------------|-----------------------------|
| 内容   | 固定题库，所有人相同   | AI 动态生成，千人千面       |
| 主题   | 通用主题               | 12 个孩子热爱的 IP 主题     |
| 评测   | 简单对错               | 阿里云专业语音评测、音素级  |
| 反馈   | 固定文案               | AI 个性化鼓励 + 发音建议   |
| 视觉   | 统一美术风格           | AI 生成主题配图 + 图生视频动画，沉浸式体验 |
| 互动   | 选择题为主             | 真实口语输出 + AI 互动故事 + AI 控制视频播放 |
